Habit & Productivity Tracker 

📌 Project Overview
The Habit & Productivity Tracker is a Python-based Command Line Interface (CLI) tool designed to help students and developers bridge the gap between physical health and academic performance.

When tackling intensive subjects like Data Structures and Algorithms (DSA) or Machine Learning, it is easy to neglect sleep and energy levels. This tool identifies the correlation between your rest and your output, providing data-driven insights to prevent burnout.

🚀 Features
1) Persistent Storage: Uses CSV file handling to save your daily logs permanently.

2) Data Integrity: Built-in error handling to prevent crashes from invalid user input.

3) Automated Insights: Calculates averages and provides "Trend Analysis" based on your habits.

4) Lightweight & Portable: No heavy databases required; runs entirely in a standard Python environment.

🛠️ Technical Concepts Used
1) File I/O: Reading and writing to .csv files using the csv and os modules.

2) Control Flow: A robust while loop menu system for a seamless user experience.

3) Data Aggregation: Using Lists and Dictionaries to calculate statistical averages.

Error Handling: try-except blocks to manage ValueError and FileNotFoundError
📦 Installation & Usage
1) Clone the Repository:
   run the following in the terminal
    git clone https://github.com/aryan2487/HabitTracker.git
    cd habit-tracker
2) Run the Application:
   Ensure you have Python 3.x installed. Run:
    python main.py
